Arquitectura AWS - Control de Flujo Vehicular


API Gateway

Es el Servicio es utilizado para ejecutar las funciones Lambda. El protocolo de comunicación utilizado es REST en formato JSON. La aplicación Angular interactúa con los demás componentes de la arquitectura por medio de API Gateway ejecutando los diferentes métodos previamente configurados


Lambda Function

Lambda es el CORE dentro de la arquitectura, es el encargado de interactuar con todos los componentes. Responde a las peticiones que se envian a los diferentes métodos de API Gateway. El lenguage de programación utilizado en las funciones Lambda es Python, ya que cuenta con un librería llamada boto3 que permite conectarse con varios servicios de AWS.


PostgreSQL RDS

RDS es el servicio de base de datos transaccionales en la nube que ofrece AWS. El motor de base de datos utilizado es PostgreSQL ya que es gratuito y de código abierto. Cuenta con todos los objetos más utilizados en bases de datos transaccionales como tablas, índices, secuencias, procesos y funciones.


AWS S3

Simple Storage Service es utilizado como File System, se almacenan los archivos de entrada que utilizan los algoritmos para realizar el análisis. Los resultados obtenidos del análisis se mueven a S3. También el servicio se utiliza como Hosting para alojar la aplicación Angular.


AWS System Manager

Es el servicio utilizado para ejecutar los algoritmos de manera remota en las maquinas virtuales. Lambda importa las funcionalidades de System Manager por medio de boto3 y hace uso del método send_command que es el encargado de enviar la secuencia de comandos de forma segura sin necesidad de usar SSH.


Media Live y Media Package

Son los servicios utilizados para la trasmisión en tiempo real. Media Live es el servicio que proporciona el canal de entrada para conectarse al live streaming usando el protocolo RTMP obteniendo como resultado un streaming de buena calidad para ser procesado por otros servicios de AWS. Media Package es el servicio que toma como fuente la salida de Media Live y empaqueta el contenido para agregar una capa de seguridad y luego ser entregado a varios dispositivos.


EC2

EC2 son las máquinas virtuales donde se ejecutan los algoritmos de análisis de información. El ambiente ya se encuentra configurado para que funcionen los algoritmos de acuerdo con la arquitectura definida. Las máquinas virtuales tienen instalado el agente que permite que System Manager ejecute comandos de manera remota.
